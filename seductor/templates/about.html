About project

Seductor
Seductor is open-source url-shortener web-application written on Python3 using Flask framework and some Flask-extensions.
URL-shortening
URL shortening is a technique on the World Wide Web in which a Uniform Resource Locator (URL) may be made substantially shorter and still direct to the required page. This is achieved by using a redirect which links to the web page that has a long URL. ref: wikipedia.org
Usage
You can create short url and QR code on main page(index page) of this site.
Limitations
On this site for unregistered users limit on generating short urls is 5 per day. Also there are some limits on creating chort links to short links. Some domains are also can not be used in long url.
Selfhosted
Service can be used on this site or deployed using your own server and domain. More information can be found in documentation.
